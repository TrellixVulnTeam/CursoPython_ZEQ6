# Anotações:
* Tipos de operadores: 
   - +  : adiçãp
   - -  : subtração
   - *  : multiplicação
   - /  : divisão com resultado em float
   - // : divisão com resultado em int
   - %  : divisão em modulo (retorna o resto)
   - ** : exponenciação
   - () : altera a precedência das operações

* Prioridade nas operações: 
   - 1 : parêntesis
   - 2 : exponenciação
   - 3 : multiplicação, divisão float e int, e módulo
   - 4 : soma e subtração
